1. When this kit is unzipped, it will be the location that the EAP, EAP Patch and DV downloads should be placed.

  2. Download the below kits to this directory location (referred to as quickstart_home).

  3. Open a terminal window and 'cd' into this quickstart_home

  4. run the setup_dv.sh (.bat) script to install and configure the DV server. This will:

    • install EAP

    • install EAP patches

    • install DV

    • start server

    • configure and deploy quick start data sources (H2 and web service)

    • deploy example VDB, name portfolio

  1. Notes on installation

    • 1 common password for all users (admin, teiidUser, dashboardAdmin): admin_1!

      • To change the password, the following 2 files need to be changed

        • eap-installer.xml.variables

        • dv-installer.xml.variables

    • to stop the server, run shutdown_server.sh

  1. Test the vdb

    • connect

      • URL: jdbc:teiid:portfolio@mm://localhost:31000

      • Username/Password: admin / admin_1!

    • Example queries

      • (h2) select * from Accounts.Customer

      • (csv) select * from Stocks.StockPrices

      • (webservice) select * from CustomerCredit.CustomersCreditView

      • (h2 & csv) select * from Stocks.StockPrices

      • (h2 & webservice) select * from CustomerCreditInfo.CustomersCreditInfoView

  1. Example Data

    • H2 database

      • Schema: Accounts

        • Table: Customer (SSN, FirstName, LastName, etc.)

        • Table: Product (ID, Symbol, Company_Name)

    • CSV file

      • Schema: MarketData

      • View: Stocks

        • Table: StockPrices (symbol, price)

    • Web service

      • Schema: CustomerCreditSource

      • View: CustomerCredit

        • Table: CustomerCreditView (customernumber, creditlimit, creditscore)

    • Views

      • Stocks.Stock: List the stockPrice for each product

      • CustomerCreditInfo.CustomersCreditInfoView: List the creditLimit and creditScore for each customer
